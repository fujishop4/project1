Tutorial

Panduan lengkap penggunaan layanan ADIJAYA VPN:

  1. Buka halaman SSH Account.
  2. Pilih server sesuai kebutuhan.
  3. Isi data akun SSH (username & password).
  4. Klik tombol Buat Akun.
  5. Simpan detail akun SSH untuk digunakan di aplikasi tunneling.

Dengan mengikuti tutorial ini, kamu bisa membuat akun SSH gratis hanya dalam beberapa menit.
