Tutorial SSH Gratis

Panduan cara menggunakan akun SSH di Android & PC dengan mudah.

ðŸ”¹ Cara Menggunakan SSH di Android

  1. Download aplikasi HTTP Injector atau HTTP Custom di Google Play Store.
  2. Buka aplikasi, lalu masuk ke menu SSH Settings.
  3. Masukkan data akun SSH yang sudah kamu buat di situs ini:
    • Host: isikan host server SSH
    • Port: 443 atau 22
    • Username: sesuai akun yang dibuat
    • Password: sesuai akun yang dibuat
  4. Simpan pengaturan, lalu klik Start.
  5. Selesai! Kini kamu bisa internetan lebih aman & cepat dengan SSH.

ðŸ”¹ Cara Menggunakan SSH di PC (Windows)

  1. Download dan install software Bitvise SSH Client atau PuTTY.
  2. Buka aplikasi, lalu masukkan data akun SSH yang sudah kamu buat:
    • Host: isikan host server SSH
    • Port: 22
    • Username: sesuai akun yang dibuat
    • Password: sesuai akun yang dibuat
  3. Klik Login untuk mulai terhubung.
  4. Jika berhasil, status koneksi akan menjadi Connected.

ðŸ’¡ Tips & Catatan

  • Gunakan server terdekat dengan lokasi kamu untuk kecepatan maksimal.
  • Jangan bagikan username & password ke orang lain.
  • Akun SSH biasanya aktif 3 - 7 hari tergantung server.
  • Jika koneksi terputus, coba buat akun baru di halaman SSH Account.
